# Calc.Basic
This is a repository for TI-84 Plus CE games
These games will ONLY work for TI-84 Plus Color Edition!
You need ALL these files for the games to work!
To get to the games, you click program, *A, then choose your game.
Games:

"AGD" - Geometry Dash, "Stereo Madness" and "Dry Out"
Enter to select level, 2nd to jump

"CE2048" - 2048
Arrow Keys, Clear to exit

"SNAKECE" - Calculator Snake
Move Left/Right/Up/Down: Use the [LEFT], [RIGHT], [UP], and [DOWN] arrow keys on the calculator's keypad to change the snake's direction.
Pause Game: Press the [ENTER] key to pause the game.
Quit Game: Press the [CLEAR] key to exit the game and return to the calculator's main menu. 

"OIRAM" - Mario
use the arrow keys for movement, the 2nd and alpha keys for jumping/sprinting, and the mode key to switch schemes. The del key is used to quit

"PacMan" - Pacman
use the arrow keys for movement, Mode to start/pause, and 2nd to exit the game by selecting "Quit"

"tetrica" - Tetris
[<] / [>] to move the piece left or right, [▲] / [2nd] to rotate clockwise, [ALPHA] to rotate counterclockwise, [v] to soft-drop, [XTΘn] to hard-drop, [MODE] to pause, and [CLEAR] to exit

"DINO" - Dino Jump
2nd to jump
